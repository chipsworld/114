

# Ultra Lightweight Dehaze Methods for Robot Vision Using High-Level Synthesis

演講日期：114/12/23
演講人：宋啟嘉 教授

關鍵字:FBGA

HLS 硬體最佳化策略
Loop Unrolling、Array Partitioning、Resource Allocation、Loop Pipelining
四個 directive 是把串列演算法轉成並行硬體的關鍵，要理解每個對吞吐量、延遲、面積的權衡。

 Dark Channel 去霧涉及除法、exposure fusion 等，用 LUT 取代倒數運算、用固定點運算來配合 FPGA 資源限制。

Transmission Map 優化 變成直接估計 air light 而不排序，降低計算複雜度。

資源占用與效能平衡LUT、BRAM、DSP 佔比，以及 full-frame processing 的 latency（8.67ms）和 FPS（115.3），顯示在有限資源下如何達到即時處理。非常成功
